Fantastic URL Shortener

Here's how it goes,

Type /url/ followed by the url you would like to shorten on to the end of this page's URL and hit enter.

For example: https://fantastic-url-shortener.glitch.me/url/https://www.google.com would return a shortened URL for https://www.google.com

Copy the shortened URL and paste it into your browser address bar to be automatically redirected to the original URL
